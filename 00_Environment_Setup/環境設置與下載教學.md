# 標準庫與編譯器安裝
1. [Arm Compiler V5 安裝](https://developer.arm.com/documentation/ka005198/1-0/)  
	下拉選擇 `Arm Compiler 5.06 update 7 (build 960)` (註冊須等一天，可找其他網路資源)  
	下載後新增資料夾 "D:\Keil_v5\ARM\ARMCC"，並解壓至此。  
   按照下面教學，完成Keil內的編譯器設定。  
   [額外載點與 Keil內設定教學](https://zhuanlan.zhihu.com/p/632808837)  

	ARM 停止對標準庫的更新，著重發展HAL庫。  
	導致 Keil 自帶的 v6 compiler 不相容下面的基本函式庫。  
   因此需要手動下載舊版編譯器，以使用標準庫。  

	"D:\Keil_v5\ARM\ARMCLANG" 相當於最新的 Arm Compiler V6。  
2. [STM32F1XX內核庫(CMSIS)與標準外設庫(StdPeriph) (包含啟動程式與暫存器定義等)](https://www.st.com/en/embedded-software/stm32-standard-peripheral-libraries/products.html)  
   下拉選擇 `STSW-STM32054`  
   教學影片的版本為 v3.5  
   自己使用 2026 年 v3.6  
   雖然 STM32F1XX 也同時對應到另一款 STSW-STM32023，  
   但是 023 是為了引導從 v2.x 升級至 v3 以上的過渡庫，不要使用。  

   ## 複製內核庫
   複製以下資料夾內的 `所有 .c & .h` 至工作目錄 (建議新增Start資料夾存放)  
   `
   "STM32F10x_StdPeriph_Lib_V3.6.0\Libraries\CMSIS\CM3\CoreSupport"  
   "STM32F10x_StdPeriph_Lib_V3.6.0\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x"  
   `

   複製以下資料夾內的 `startup_stm32f10x_md.s` 至工作目錄 (建議新增Start資料夾存放)  
   md 代表 STM32F103C`8`t6 對應到 64 KB Flash 為中容量型號 (見參考手冊)  
   _vl 代表 F100 的閹割版系列，因此不須使用。 
   `
   "STM32F10x_StdPeriph_Lib_V3.6.0\Libraries\CMSIS\CM3\DeviceSupport\ST\STM32F10x\startup\arm"  
   `

   ## 複製標準外設庫
   複製以下資料夾當中的所有檔案 至工作目錄 (建議新增Library資料夾存放)  
   庫函數的 .c (src) 與 .h (inc)  
   `
   "STM32F10x_StdPeriph_Lib_V3.6.0\Libraries\STM32F10x_StdPeriph_Driver\inc"  
   "STM32F10x_StdPeriph_Lib_V3.6.0\Libraries\STM32F10x_StdPeriph_Driver\src"  
   `

   複製以下檔案 至工作目錄 (本專案存放於User資料夾)  
   處理庫函數 .h 的包含關係 & 參數檢查定義  
   `
   "STM32F10x_StdPeriph_Lib_V3.6.0\Project\STM32F10x_StdPeriph_Template\stm32f10x_conf.h"  
   `

   複製以下檔案 至工作目錄 (建議與main.c相同位置)  
   中斷函數  
   `
   "STM32F10x_StdPeriph_Lib_V3.6.0\Project\STM32F10x_StdPeriph_Template\stm32f10x_it.c"  
   "STM32F10x_StdPeriph_Lib_V3.6.0\Project\STM32F10x_StdPeriph_Template\stm32f10x_it.h"  
   `

# Keil 基本設定
1. 工具列, edit, configuration, editor, encoding, ANSI -> UTF-8 // 支援中文  
2. 工具列, edit, configuration, editor, C/C++ files, Tab Size, 2 > 4 // 縮排大小  
3. 工具列, edit, configuration, color & fonts, ASM, font, size, 10 -> 12 // 字體大小  
4. 工具列, edit, configuration, color & fonts, C/C++, font, size, 10 -> 12 // 字體大小 

# Keil 專案設定 (每個新專案都要重設) 
1. 魔法棒(Option for target), Debug, U-link > ST-link // 設置調適器  
2. 魔法棒(Option for target), Debug, ST-link, Settings, Flash Download, Reset & Run > 打勾 // 燒錄後自動復位執行  
3. 魔法棒(Option for target), Debug, ST-link, Settings, Pack, 取消勾選Enable // 新版 Keil (5.28+) 需要加上此設定，才能自動復位
4. 魔法棒(Option for target), Debug, Dialog DLL, DCM.DLL -> DARMSTM.DLL & -pCM3 -> -pSTM32F103C8 & TCM.DLL -> TARMSTM.DLL & -pCM3 -> -pSTM32F103C8 // Debug當中 啟用圖形化的Peripheral變數監看 (Dialog DLL)

## 以下重要專案設定 不設會報錯
4. 魔法棒(Option for target), C/C++, Include Paths, ..., 虛線白紙的icon(New), ..., 設定上面基本庫的路徑(Start, Library, User)  
5. 開啟 "stm32f10x.h" 拉到最下面，L8341~L8343，貼到main.c當中。  
   ```c
   #ifdef USE_STDPERIPH_DRIVER // 如果設定使用標準庫 才進行引用 設定方式參考下一步
      #include "stm32f10x_conf.h"
   #endif
   ```
6. 魔法棒(Option for target), C/C++, Preprocessor Symbols, Define, 貼上 `USE_STDPERIPH_DRIVER`
